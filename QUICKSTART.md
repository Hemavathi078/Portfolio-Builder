## üéØ First Time User Flow

### 1. Landing Page
- You'll see a beautiful landing page
- Click **"Build Your Resume"** for job applications
- Or click **"Build Your Portfolio"** for showcasing work

### 2. Fill Your Information
Navigate through the sidebar:

**Personal Info** ‚Üí Add your name, title, contact, photo, summary

**Auto Summary** ‚Üí Generate professional summary:
- Select your role (Frontend Developer, etc.)
- Choose experience level
- Click "Generate Summary"
- Edit as needed

**Education** ‚Üí Add your degrees:
- Click "Add Education"
- Fill institution, degree, years
- Add multiple entries

**Experience** ‚Üí Add work history:
- Click "Add Experience"
- Fill company, role, duration, responsibilities
- Add multiple entries

**Skills** ‚Üí Choose format:
- **Visual Skills** (for portfolio): Progress bars
- **Categorized Skills** (for resume): Bullet points
  - Technical Skills: JavaScript, React, etc.
  - Tools: Git, VS Code, etc.
  - Soft Skills: Communication, etc.

**Projects** ‚Üí Showcase your work:
- Click "Add Project"
- Add title, description, tech stack
- Add GitHub and live URLs
- Upload project image

**Reference Upload** (Optional):
- Upload a sample resume format
- Use as visual reference
- Show/hide as needed

**Theme** ‚Üí Customize appearance:
- Choose gradient preset
- Select font (ATS-safe for resumes)
- Adjust font size
- Toggle section visibility
- Enable dark mode

### 3. Preview Your Work
- Click **"Resume"** to see ATS-friendly resume
- Click **"Portfolio"** to see visual portfolio
- Changes update in real-time

### 4. Export as PDF
- Click "Export as PDF" button
- PDF downloads automatically
- Ready to send to employers!

## üí° Pro Tips

### For Job Applications (Resume Mode)
1. Use **ATS-safe fonts**: Inter, Roboto, or Arial
2. Fill **Categorized Skills** (not visual)
3. Use **Auto Summary** for professional tone
4. Keep it **single-column** (automatic)
5. Use **standard section headings** (automatic)
6. Export as PDF and apply!

### For Portfolio Websites
1. Use **Visual Skills** with progress bars
2. Add **project images** for visual appeal
3. Choose **colorful gradients**
4. Enable **glassmorphism** card style
5. Add **GitHub and live URLs**
6. Share the PDF or deploy as website

### General Tips
1. **Auto-save**: Your data saves automatically
2. **Reference Upload**: Upload a resume you like as reference
3. **Section Visibility**: Hide sections you don't need
4. **Dark Mode**: Toggle for comfortable editing
5. **Back to Home**: Return to landing page anytime
6. **Reset Data**: Start fresh when needed

## üêõ Troubleshooting

### PDF Export Not Working
- Ensure preview is fully loaded
- Check browser console for errors
- Try disabling browser extensions

### Data Not Saving
- Check browser localStorage is enabled
- Clear browser cache
- Try incognito mode

### Preview Not Updating
- Refresh the page
- Check if you're in the correct preview mode
- Verify data is entered in forms

### File Upload Issues
- Check file size (max 5MB)
- Use supported formats: JPG, PNG, PDF
- Try a different file
